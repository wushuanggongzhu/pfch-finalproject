Visualizing Acquisitions: The Development of the MET's China Collection, 1875-2023

By: Lily Amelia Susman

  • Launch Project
  • View on Github
Completed in Spring 2023 for INFO 664: Programming for Cultural Heritage, "Visualizing Acquisitions: The Development of the MET's China Collection, 1875-2023" is uses data from the Metropolitan Museum of Art. The project consists of a dashboard on Tableau Public and a Github repository. The data set, which is foundational to the visualization, was built through two Python scripts. Communicating with the MET’s API, I specified search parameters and informational fields that I sought to capture. For this project, I wanted catalog data for all of the museum’s Chinese works. After writing the data into a CSV, I imported it into Tableau. There I created two visualizations. Filterable by art classification (e.g. medium or materials) and year of acquisition, the visualizations display the collection’s growth and its composition.
